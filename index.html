From Hypertext to the World Wide Web

In 1989, a simple proposal at CERN changed the world forever. Tim Berners-Lee imagined a system where documents could link to other documents — turning a research network into the global information web we know today.

This site explores how hypertext became the foundation of the World Wide Web, and why it remains one of the most important inventions of the modern era.

Core Concepts

🔗

Hypertext

Text that contains links to other texts, enabling non-linear reading and navigation across documents stored anywhere on the web.

🌐

The Web vs. The Internet

The Internet is the infrastructure (cables, routers, protocols). The Web is a service built on top of it — a collection of linked hypertext pages.

📄

HTML

HyperText Markup Language is the standard language for creating web pages, using hyperlinks to connect documents across the globe.

📡

HTTP

HyperText Transfer Protocol governs how hypertext documents are transmitted between a web server and a browser.

📍

URL

A Uniform Resource Locator is a web address that uniquely identifies the location of a hypertext resource on the internet.

🖥️

Web Browser

Software (like Chrome or Firefox) that retrieves, renders, and navigates hypertext documents by interpreting HTML and CSS.

Why Hypertext Changed Everything

Before the Web, information was siloed. Academic papers, manuals, and databases existed in isolation. You couldn't follow a reference from one document directly into another; you had to manually locate it, often in a different building or library.

"The Web is more a social creation than a technical one. I designed it for a social effect — to help people work together — and not as a technical toy." — Tim Berners-Lee

Hypertext solved this by embedding navigable links directly within content. A reader could jump from a concept to its definition, from a citation to its source, and from a website to any other website on the planet — instantly.

This simple idea — documents that link to other documents — became the architectural backbone of the modern Web and enabled everything from Wikipedia to e-commerce to social media.

Quick Facts

  • Ted Nelson coined the term "hypertext" in 1963
  • Berners-Lee wrote the first proposal at CERN in 1989
  • First web server ran on a NeXT computer
  • The first website is still online at info.cern.ch
  • HTTP/1.1 was standardised in 1997
  • HTML5 was finalised in 2014
  • Over 1.9 billion websites exist today

Hypertext vs. Traditional Text

Feature Traditional Text Hypertext
Navigation Linear — read from start to finish Non-linear — follow links in any direction
Cross-references Manual — "see page 47" or a bibliography Instant — click a link to jump to any resource
Media Primarily text and static images Text, images, video, audio, interactive elements
Location Physical medium (book, printout) Distributed across servers worldwide
Updating Requires reprinting or revision Updated instantly on the server
Accessibility Limited by geography and physical copies Available anywhere with internet access
